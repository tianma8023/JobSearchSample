# JobSearchSample
a simple demo based on graphql api: https://api.graphql.jobs

## Info
- architecture: MVVM
- detailed information
    - Database & ORM: [Room](https://developer.android.com/jetpack/androidx/releases/room)
    - Network request: [Apollo](https://github.com/apollographql/apollo-kotlin) & [OkHttp](https://github.com/square/okhttp)
    - Image loading library: [coil](https://github.com/coil-kt/coil)
    - Fragment navigation: [Navigation](https://developer.android.com/jetpack/androidx/releases/navigation)
    - Coroutine for Lifecycle: [kotlinx.coroutines](https://github.com/Kotlin/kotlinx.coroutines) & [Lifecycle](https://developer.android.com/jetpack/androidx/releases/lifecycle)
    - UI layer library: [ViewBinding](https://developer.android.com/topic/libraries/view-binding)
